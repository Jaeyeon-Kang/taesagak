# 퇴사각 — 근로자용 돈찾기 계산기 MVP

정적 웹앱이다.  
한 번 입력하면 다음 4가지를 같이 본다.

- 퇴직금 간편 추정
- 주휴수당 대상 여부 및 월 환산액
- 세후 월급 간이 추정
- 최저임금 미달 가능성 점검

앱의 목적은 “숫자 하나 더 보여주기”가 아니다.  
**숫자 + 계산 근거 + 확인할 항목**을 같이 보여줘서 사용자가 다시 검색창으로 되돌아가는 일을 줄이는 데 있다.

---

## 1. 스택

- HTML
- CSS
- Vanilla JavaScript (ES modules)
- PWA manifest + service worker
- 로컬 저장(localStorage)
- 빌드 의존성 없음

정적 호스팅에 바로 올릴 수 있게 만들었다. Cloudflare Pages 같은 데 딱 맞는다.

---

## 2. 실행 방법

### 가장 쉬운 방법
프로젝트 루트에서 간단한 HTTP 서버를 띄운다.

```bash
python -m http.server 8080
```

브라우저에서 아래 주소로 들어간다.

```text
http://localhost:8080
```

`file://`로 직접 열어도 일부 브라우저에선 동작하긴 하지만,
서비스워커와 모듈 로딩까지 생각하면 로컬 서버로 보는 편이 낫다.

## 4. 계산 로직 요약

### 주휴수당
- 기준: 주 15시간 이상 + 개근 가정
- 간편식:
  - 1일 소정근로시간 = 주간 총근로시간 ÷ 주 근무일수
  - 주휴수당(주) = 1일 소정근로시간 × 시급

### 퇴직금
- 기준:
  - 1년 이상 계속근로
  - 주 평균 15시간 이상
- 간편식:
  - 법정퇴직금 = `max(1일 평균임금, 1일 통상임금) × 30 × (총 재직일수 ÷ 365)`

### 세후 월급 추정
- 목적: 급여명세서와 1:1 일치시키는 기능이 아니라 “대략 어디서 깎이는지” 빠르게 파악하는 기능
- 포함:
  - 국민연금
  - 건강보험
  - 장기요양보험
  - 고용보험
  - 근로소득공제
  - 기본공제
  - 근로소득세액공제
  - 지방소득세

### 최저임금 점검
- 2026년 기준 시간급 최저임금으로 비교
- 월급 입력 시에는 월 환산 유급시간으로 역산해 시급을 만든다

## 5. 주의

이 MVP는 **법률/세무 자문 도구가 아니다**.

특히 아래 항목이 들어오면 실제 값과 차이가 날 수 있다.

- 비과세 식대/수당
- 회사별 급여 항목 처리
- 월중 입·퇴사
- 상여/연차수당 산입 방식
- 국민연금 예외 대상
- 공제대상 가족 / 특별공제 / 간이세액표 반영 차이

즉, 이 앱은 “1차 점검 + 행동 유도” 용도다.
